# -FE25-JS2-slutprojekt-back-Mohsen-Heidari

Detta är backend-delen av projektet Scrum Board. Backend är byggd med Node.js, Express och TypeScript och används för att hantera medlemmar och uppgifter (assignments) i projektet.

API:et lagrar data i Firebase Realtime Database och gör det möjligt att skapa, läsa, uppdatera och radera medlemmar och uppgifter.
